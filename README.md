
# Simple To-Do List Python Project

This is a simple console-based To-Do List application implemented in Python. The project serves as a basic demonstration of fundamental Python concepts such as classes, methods, loops, and conditional statements.

## Features

- Add tasks to the To-Do List.
- View the existing tasks in the To-Do List.
- Remove tasks from the To-Do List based on their index.
- Quit the To-Do List application.
